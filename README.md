# 🌍 Language69 Translator Bot

A powerful Telegram bot that translates text between 69 languages with auto-detection.

## 🚀 Features

- ✅ Auto-detects source language
- ✅ Supports 69 languages
- ✅ User-specific target language preferences
- ✅ Inline keyboard interface
- ✅ Fast and accurate translations
- ✅ 24/7 uptime via Railway

## 🛠️ Commands

- `/start` - Welcome message
- `/help` - Usage instructions
- `/about` - Bot information
- `/langs` - List all 69 languages
- `/setlang` - Set your target language

## 📦 Deployment

This bot is deployed on Railway with GitHub integration.

### Environment Variables

- `BOT_TOKEN` - Your Telegram bot token from @BotFather

## 🔧 Local Development

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set `BOT_TOKEN` environment variable
4. Run: `python bot.py`

## 📊 Tech Stack

- Python 3.11
- python-telegram-bot
- googletrans
- Railway
